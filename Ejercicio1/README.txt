El ejercicio se compila con:
gcc Comentarios_C.c -o comentarios.out

Para ingresar datos al sistema desde teclado se utiliza
./comentarios.out
Despues de escribrir la cadena pulsa ENTER para ingresar los datos al sistema.

Pulsa Ctrl+D para salir del sistema.

Para correr los programas usando un archivo de texto con las cadenas:
./comentarios.out < comentarios.txt
NOTA: Si se desea visualizar las cadenas provenientes del archivo comentarios.txt será necesario descomentar la lína 65.
